🛒 AtoZ Cart – Online Order Collection Website

AtoZ Cart is a modern, responsive eCommerce-style website that allows users to browse products, add items to a cart, and place orders easily through a simple checkout form.

---

🚀 Features

- 🛍️ Product listing with categories
- 🔍 Filter products by category
- 🛒 Add to cart functionality
- ➕ Increase/decrease product quantity
- ❌ Remove items from cart
- 💾 Cart saved using Local Storage
- 📦 Order form with customer details
- 📲 WhatsApp integration for direct contact
- 📱 Fully responsive design

📦 How It Works

1. Users browse products on the homepage
2. Click "Add to Cart"
3. Open cart and manage items
4. Click "Place Order"
5. Fill out the form
6. Order is submitted via Formspree
7. Admin contacts user via WhatsApp

---

🔧 Customization

You can easily customize:

- 🧾 Products → Edit inside "script.js"
- 🎨 Styles → Modify "style.css"
- 📞 Contact Info → Update in "index.html"
- 📩 Form Endpoint → Replace Formspree link
